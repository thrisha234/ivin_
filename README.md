# Live Meeting Transcription & Speaker Recognition


## Setup


1. Create a Python virtual environment and activate it.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set `OPENAI_API_KEY` environment variable if you want to use OpenAI APIs for summarization (optional).
4. Run: `python app.py` (development) or use a WSGI server for production.


Access `http://localhost:5000`.
